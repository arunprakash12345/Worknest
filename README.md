🚀 WorkNest – Student Task Management Platform

A full-stack task management platform designed for mentors and students to collaborate, track progress, manage assignments, and streamline cohort workflows.

✨ Features
🔐 JWT Authentication & Role-Based Access Control
👥 Batch Creation & Member Management
✅ Task Creation, Assignment & Progress Tracking
💬 Task Discussions (GitHub, PR, Figma & Feedback Links)
📅 Interactive Task Calendar
📊 Analytics Dashboard
🎯 Personalized "My Tasks" Workspace
🌙 Dark / Light Theme Support
🔒 Secure Resource Access & Authorization
🎨 UX Principles
Role-based experiences for Mentors and Students
Reduced cognitive load through progressive disclosure
Context-aware navigation and task workflows
Secure and intuitive collaboration experience
Responsive design across devices
